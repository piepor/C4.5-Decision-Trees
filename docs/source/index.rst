Welcome to c4dot5's documentation!
==================================
This section contains documentation on how to use the different features of the c4dot5 package.
c4dot5 lets you train a decision tree using Quinlan's C4.5 algorithm. Of course, it offers others features such as inference, visualization and rules extraction.
